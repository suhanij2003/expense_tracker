# Expense Tracker

A modern React-based dashboard application built using Vite.

##  Features
- Authentication system
- Protected routes
- Dashboard analytics
- Product management UI
- Theme switching (Dark/Light)
- Responsive design
- Skeleton loaders for better UX

##  Tech Stack
- React.js
- Vite
- JavaScript (ES6+)
- CSS
- Context API

##  Project Structure
src/
 ├── components/
 ├── pages/
 ├── context/
 ├── assets/

## Getting Started

```bash
npm install
npm run dev
